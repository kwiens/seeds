# Seeds — Chattanooga National Park City

A community platform for proposing and supporting local projects ("seeds") that help Chattanooga grow as a National Park City. Browse ideas on a grid or map, support the ones you love, and watch them grow.

## Features

- **Plant a Seed** — Submit community project proposals with details like gardeners, organizations, resources, and location
- **Explore** — Browse seeds in a card grid or interactive Mapbox map with category filters
- **Support** — Sign in with Google and give "sunlight" to seeds you believe in
- **AI Illustrations** — Each seed gets a unique watercolor-style illustration generated by Google Gemini, stored in Vercel Blob
- **Admin Dashboard** — Approve, reject, or manage submitted seeds
- **User Dashboard** — Track your seeds and their supporters

## Tech Stack

- **Framework**: Next.js 16 (App Router, React 19)
- **Language**: TypeScript (strict mode)
- **Styling**: Tailwind CSS v4 + shadcn/ui
- **Database**: Neon PostgreSQL + Drizzle ORM
- **Auth**: NextAuth.js v5 (Google OAuth)
- **Maps**: Mapbox GL JS via react-map-gl
- **AI Images**: Google Gemini (`gemini-2.5-flash-image`)
- **Storage**: Vercel Blob
- **Deployment**: Vercel

## Getting Started

```bash
# Install dependencies
pnpm install

# Pull environment variables from Vercel
vercel env pull

# Push DB schema (first time)
pnpm db:push

# Start dev server
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment Variables

Copy `.env.example` and fill in the values:

| Variable | Description |
|----------|-------------|
| `DATABASE_URL` | Neon PostgreSQL connection string |
| `AUTH_SECRET` | NextAuth secret (`openssl rand -base64 32`) |
| `AUTH_GOOGLE_ID` | Google OAuth client ID |
| `AUTH_GOOGLE_SECRET` | Google OAuth client secret |
| `NEXT_PUBLIC_MAPBOX_TOKEN` | Mapbox public access token |
| `ADMIN_EMAILS` | Comma-separated admin emails (auto-promoted on sign-in) |
| `GOOGLE_GENERATIVE_AI_API_KEY` | Google Gemini API key |
| `BLOB_READ_WRITE_TOKEN` | Vercel Blob read/write token |
